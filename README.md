# wow-autofish
魔兽世界自动钓鱼插件    
# 项目结构
GetOpenFileNameTest 插件主项目代码（知道你想吐槽项目名字。。。我接受囧！）
LicGenerator key生成器
# 插件实现原理
主要通过图像识别和语音识别技术，启动软件后，会扫描一定区域的鼠标指针形状，如果指针变成鱼钩形状，则等待鱼上钩的声音，触发收竿动作。软件未使用任何wow相关脚本、dll注入、内存修改等外挂技术，完全模拟真人操作，使用相对安全。
